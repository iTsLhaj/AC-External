# AC-External

![](https://i.pinimg.com/736x/8f/f9/8a/8ff98a53f6ecf8abca1365366ef09f02.jpg)

Small archived experiment in external memory manipulation for a game.  
Reads process memory, locates player data, and applies simple value edits.  
Made while learning Windows APIs, pointer scanning, and basic reversing.

For educational use only.
